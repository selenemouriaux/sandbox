définition

un sélécteur cible un ou plusieurs élements html

  • nom de balise
  • nom de classe
  • nom de id
  • nom de la pseudo classe :
    • :visited, :hover, :link
    • :nth-child
    • :nth-of-type
    • :first-child
    • :last-child
    • ...

utilisation

pour combiner les composants d'un sélecteur on a plusieurs caractères :

  • espace (descendant)
  • > (enfant de premier niveau)
  • + (le premier des éléments suivants)
  • ~ (tous les éléments suivants)
  • * (tout)
